# Governanca de Dados / Data Governance


Este documento define diretrizes de governanca de dados no **GOV-GB**.

### Objetivos

- Assegurar qualidade, integridade e rastreabilidade dos dados.
- Definir responsabilidades por ciclo de vida dos dados.
- Reduzir risco operacional, legal e reputacional.

### Classificacao de Dados

- **Publico**: pode ser compartilhado sem restricoes.
- **Interno**: uso interno do projeto e parceiros autorizados.
- **Sensivel**: requer controles reforcados de acesso e auditoria.

### Responsabilidades

- **Data Owner**: define regras de uso e qualidade.
- **Data Steward**: garante padroes e documentacao.
- **Data Consumer**: utiliza dados conforme politica vigente.

### Qualidade de Dados

- Definir validacoes de formato e consistencia.
- Medir completude e atualizacao.
- Corrigir anomalias com trilha de auditoria.

### Acesso e Controle

- Acesso por papel e principio de menor privilegio.
- Revisao periodica de permissoes.
- Registro de acessos e operacoes sensiveis.

### Continuidade e Resiliencia

- Politica de backup e recuperacao testada.
- Procedimentos de restauracao documentados.
- Plano de continuidade para servicos criticos.
